• Data is current as of May 2013

Scientific Impact metrics are calculated using only publications that XSEDE users have specifically associated with a project using the tools provided by XSEDE User Portal. While this represents only a subset of a user’s total publications, it is not currently feasible to automatically make the association between a user’s publication and an XSEDE project. 


    Metrics By User are calculated per user using the publications associated with each user
    Metrics by Organization are calculated per organization using publications aggregated over all users associated with a particular organization in May 2013. Note that organizational affiliations prior to May 2013 are not currently taken into account.
    Metrics by Project are calculated on a project basis using publications that have been identified as associated with a given project when entered by a user. Projects may be active or inactive.

Metric Name

Metric Description
# of Pubs

 Number of publications associated with the user/organization/project, since 2008
Cited by Citations have received so far of ALL the publications, since 2008
h-index The h-index computed across all available publications, since 2008. The h-index is the maximum number h such that h publications associated with the entity received at least h citations each.
See http://en.wikipedia.org/wiki/H-index and http://www.pnas.org/content/102/46/16569
g-index The g-index computed across all available publications, since 2008. The g-index is the unique largest number such that the top g publications received at least g^2 citations.
See http://en.wikipedia.org/wiki/G-index and http://link.springer.com/article/10.1007%2Fs11192-006-0144-7

i10-index

 The i10-index computed by across all available publications, since 2008. The i10-index is the number of publications that have received at least 10 citations, for a given entity. Google Scholar introduced this metric first in 2011.
See http://en.wikipedia.org/wiki/I10-index
